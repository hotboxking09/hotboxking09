<h1 align="center">JARVIS SYSTEM</h1>
<p align="center"><strong>Independent Operating System Project · Switzerland</strong></p>

## Grüezi, ich bin Oliver

Ich bin **Oliver-Frank Pristaff**, Gründer und Entwickler von **JARVIS SYSTEM**. Ich entwickle eine eigenständige Systemplattform mit klarer technischer Identität – von der UEFI-Bootoberfläche über Anmeldung und Unified Shell bis zur echten Systemtelemetrie.

> JARVIS OS ist aktiv in Entwicklung. Die aktuelle Grundlage nutzt weiterhin den Linux-Kernel und Linux-Treiber. Ein eigener fertiger Kernel oder eine öffentliche ISO werden nicht vorgetäuscht und derzeit nicht veröffentlicht.

### Engineering Focus

- **UEFI & Boot** — eigene sichtbare Bootumgebung und dynamische Systemerkennung
- **System Runtime** — JARVIS Login, Unified Shell, Frame Core und Systemprogramme
- **Real Telemetry** — reale Hardwaredaten; nicht messbare Werte bleiben `N/A`
- **Portability** — VM-Tests und schrittweise Validierung auf unterschiedlicher realer Hardware
- **Documentation** — nachvollziehbare Builds, Tests, Entscheidungen und Entwicklungsberichte
